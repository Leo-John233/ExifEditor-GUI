## 📸 EXIF 迁移与修改工具 (EXIF Sync & Edit GUI)

这是一个基于 Python 和 tkinter 构建的轻量级图形界面工具，核心由 ExifTool 驱动，本工具专为摄影师和后期爱好者设计，主要解决在“RAW 转 JPG”或“跨相机多机位拍摄”工作流中，照片 EXIF 数据（尤其是 GPS 定位和镜头信息）丢失、错乱，或导致手机相册无法正常识别地理位置的痛点

## ✨ 核心特性

- * 📱 完美适配手机相册 (纯净迁移)：默认剔除臃肿的相机厂商私有数据 (--MakerNotes)，严格遵循国际标准重写 EXIF，彻底解决“用 PS/Lr 导出后自己补全 GPS，但传到 iPhone/安卓相册却无法显示定位”的问题

- * 🔄 双模式自由切换：
*1*. 数据迁移模式：从 RAW/JPG 中无损提取全部环境参数，一键注入到目标后期出图 (JPG/TIFF) 中
*2*. 直接修改模式：不依赖源文件，直接对单张照片进行参数覆写（如补全老镜头的无触点信息）

- * ⏰ 时间修改：可以同时通过日历选择的方式或者直接输入的方式修改照片的拍摄时间

- * 🧳 开箱即用的“绿色便携版”体验：内置智能路径探测逻辑，打包为 .exe 后，只需将 exiftool.exe 放在软件同级目录，即可在任何 Windows 电脑上免安装直接运行

- * 🎛️ 丰富的可视化修改面板：支持 4 大类、超 20 项核心 EXIF 参数的可视化编辑（涵盖创作者版权、相机镜头、曝光参数、精确时间与经纬度等）

## 🛠️ 安装与使用
### 方法一：使用打包好的便携版 (推荐)
如果你只是想快速使用这款软件，不需要配置任何 Python 环境：
- (1). 在本仓库的 Releases 页面下载最新打包好的 `.exe` 软件

- (2). 前往 ExifTool 官网 下载 Windows 版本的独立可执行文件（解压后将名字重命名为 `exiftool.exe`）

- (3). 将下载好的两个 `.exe` 文件放在同一个文件夹下

- (4). 双击运行软件，即可直接使用
### 方法二：通过 Python 源码运行
如果你想自行修改代码或打包：
- (1). 克隆本仓库到本地：
```cpp
git clone https://github.com/Leo-John233/ExifEditor-GUI
```
- (2). 确保你的电脑已安装 `Python 3.x`（本程序仅使用了 Python 标准库 `tkinter`、`subprocess`、`os` 等，无需 `pip install` 任何第三方模块）

- (3). 准备好 `exiftool.exe`，并在代码界面中指定其路径

- (4). 运行脚本：
```cpp
python exiftool_gui.py
```
